2. WHat is typescript?

  • Microsoft 's solution to large-scale Javascript application development:
Typed superset of Javascript that compiles to plain Javascript
Formalizes a type system that describes Javascript dynamic types in real-(development)-time

2. WHat is typescript?

  • Microsoft 's solution to large-scale Javascript application development:
Better tooling support  for large-scale Javascript development
TypeScript is  Contract-Oriented Programming  /  Intent-Oriented Programming
